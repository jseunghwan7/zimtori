# 짐토리 Codex 작업용 소스

현재 짐토리 웹앱의 화면, 로컬 체험 기능, 브랜드 이미지가 포함된 독립 실행 프로젝트입니다.
ChatGPT Sites 전용 배포 설정과 설치 캐시는 제외했으므로 Codex에서 수정하고 로컬 미리보기하기에 적합합니다.

## 시작하기

Node.js 20.9 이상과 npm이 필요합니다.

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 엽니다.

## Codex에 요청하는 예시

```text
이 프로젝트를 먼저 실행해서 현재 화면을 확인해줘.
짐토리 옷입히기 화면의 선글라스를 눈 위치에 맞게 조정하고,
모바일과 데스크톱에서 확인해줘. 배포는 하지 마.
```

## 주요 파일

- `app/zimtori-app.tsx`: 화면과 사용자 흐름, 로컬 데이터 기능
- `app/globals.css`: 전체 디자인과 반응형 스타일
- `public/assets/`: 햄스터, 로고, 옷, 캐비닛, 대여 물품 이미지
- `app/layout.tsx`: 앱 제목, 아이콘, 모바일 뷰포트 설정

## 현재 데이터 방식

- 신청·코디·이용내역: 브라우저 `localStorage`
- 위탁 물품 사진: 브라우저 `IndexedDB`
- Google·카카오 로그인, 카카오 지도, 결제, 서버 DB: 아직 연결하지 않음

## 배포

수정이 완료되기 전에는 `npm run dev`로만 확인하는 것을 권장합니다.
최종 단계에서 Netlify 또는 다른 호스팅 환경에 맞춰 별도로 설정할 수 있습니다.
